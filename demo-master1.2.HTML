The illusion of non-linearity

M1.2 - Design Research Project DEMO

This project had two versions. One of the versions featured a linear game, where players had to use collect keys at the end of each level to open the gate to the next level. The other version included the “illusion of non-linearity”, in this version every level was accessible, but the first level entered would always load the first level, the second level entered would load the second level and so on.

Both versions are playable by downloading the game files.

Download instructions:

The download button on this page will open a google drive folder feature 3 zip files. These zip files contain the game files. You can download whichever version of the game you want to try.

Illusion – featuring the illusion of non-linearity

Linear – featuring linear levels

Random – randomly picks one of the two when starting to play

Download the zip file and unzip it somewhere on your computer to play. Play by opening “game.exe”.
